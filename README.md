# IPB Food UMKM

IPB Food UMKM adalah Platform informasi kantin dan UMKM di lingkungan kampus IPB.

## 📂 Struktur Proyek
- **frontend/**: Aplikasi antarmuka menggunakan React + Vite + Tailwind.
- **backend/**: Layanan API dan pengolahan data (FastAPI).

## 🚀 Cara Menjalankan Proyek Lokal

### Frontend
1. Masuk ke folder: `cd frontend`
2. Install library: `npm install`
3. Jalankan: `npm run dev`

### Backend
1. Masuk ke folder: `cd backend`
